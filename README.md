# Npm-TypeScript-Hono

## .env は、各自設定(default 3001)
~~~env
HONO_PORT=***
~~~

## Build & Run
```bash
make br
```

## Accessed by browser
~~~bash
http://localhost:3001/
~~~

### Response
~~~bash
Hello World!
~~~

## Accessed by browser
~~~bash
http://localhost:3001/ping
~~~

### Response
~~~bash
pong
~~~
